Quick Command Reference
=======================

Deploy (from Mac):
  cd /Users/maxwraae/picar-setup/picar-brain
  ./deploy_to_pi.sh

Setup (on Pi):
  cd ~/picar-brain
  ./setup_voice_assistant.sh

Test (on Pi):
  cd ~/picar-brain
  ./test_voice_assistant.sh

Run (on Pi):
  cd ~/picar-brain
  source venv/bin/activate
  python3 voice_assistant.py

Architecture Flow
=================

┌─────────────┐
│   Listen    │ ← speech_recognition + Whisper API
└──────┬──────┘
       │
┌──────▼──────┐
│   Think     │ ← OpenAI Chat Completions (GPT-4)
└──────┬──────┘   + conversation history
       │
┌──────▼──────┐
│   Speak     │ ← Piper TTS (Swedish)
└──────┬──────┘
       │
┌──────▼──────┐
│    Act      │ ← PiCar movements
└─────────────┘

Key Features
============

✓ Chat Completions API (stable, long-term)
✓ Conversation memory (last 10 exchanges)
✓ Piper TTS (free, offline, Swedish)
✓ Fun robot personality for Leon (9 years)
✓ Movement actions (spin, dance, nod, shake)
✓ Simple architecture (linear loop)
✓ Easy to maintain and modify
✓ Fully documented
